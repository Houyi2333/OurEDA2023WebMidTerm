## 作业要求

1. 请自行创建 React 项目完成作业，请遵守模块化设计理念，合理设计功能模块，并规范文件和文件夹命名
2. 请使用 React 的**函数式组件**进行代码编写，使用 `useState()`，`useEffect()`等钩子函数引入 React 特性
3. 作业**全过程**请使用 Git 进行版本控制
4. 请严格规范自己的代码风格，包括缩进、变量命名、函数命名等，尽可能使代码整洁而具有较高可读性
